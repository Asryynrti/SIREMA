    Langkah - langkah untuk mengetahui rekomendasi mata kuliah pilihan:
  • Memilih KBK yang ada di bagian Pilih KBK.
    		•
  • Input nilai-nilai mata kuliah prasyarat sesuai dengan KBK yang dipilih.
    		•
  • Tekan tombol "Prediksi".
    		•
  • Keterangan mata kuliah pilihan dalam KBK:
    • KBK Telekomunikasi = Sistem dan Jaringan Telekominikasi.
    • KBK Sistem Informasi Bisnis = Konsep E-Bisnis.
    • KBK Teknik Informatika = Pemrograman Internet.
    • KBK Vokasi Informatika = Computer-Based Learning.
  • Keterangan hasil:
    • Cluster 0 = Direkomendasikan.
    • Cluster 1 = Tidak Direkomendasikan.
